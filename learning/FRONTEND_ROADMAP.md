## Phase 01: Web Fundamentals (🌍)
**Duration:** 2–3 weeks  
**Color:** #ff6b35  
*HTML, CSS, and the basics. Every pixel on the web starts here.*

### Topic 1.1: HTML Essentials
**Difficulty:** Beginner | **Tag:** lang  
*Semantic markup, forms, accessibility. Structure before style.*

- HTML document structure and doctypes
- Semantic HTML5 elements (header, nav, main, article, footer)
- Form elements and validation
- Attributes and metadata
- SEO basics and microdata

### Topic 1.2: CSS Fundamentals
**Difficulty:** Beginner | **Tag:** lang  
*Selectors, cascade, specificity, box model. Foundations of styling.*

- Selectors (element, class, ID, pseudo-classes, pseudo-elements)
- Cascade and specificity
- Box model (margin, padding, border, content)
- Units (px, em, rem, vh, vw)
- Color and typography

### Topic 1.3: CSS Layout
**Difficulty:** Beginner | **Tag:** skill  
*Display, positioning, flexbox intuition.*

- Display property (block, inline, inline-block)
- Positioning (static, relative, absolute, fixed, sticky)
- Float and clearfix
- Margins and collapsing
- Stacking context and z-index

### Topic 1.4: Flexbox
**Difficulty:** Easy | **Tag:** skill  
*Modern 1D layout system. Alignment, distribution, flexibility.*

- Flex container and flex items
- Main axis and cross axis
- Flex properties (flex-grow, flex-shrink, flex-basis)
- Justify-content and align-items
- Building simple responsive layouts

### Topic 1.5: CSS Grid
**Difficulty:** Easy | **Tag:** skill  
*2D layout powerhouse. Rows, columns, gaps, areas.*

- Grid container and grid tracks
- Grid lines and grid cells
- Grid template areas
- Auto-placement and flow
- Grid vs Flexbox decision-making

### Topic 1.6: Responsive Design
**Difficulty:** Easy | **Tag:** skill  
*Mobile-first, media queries, viewport meta tag.*

- Viewport meta tag and device pixels
- Media queries and breakpoints
- Mobile-first approach
- Responsive images and srcset
- Touch-friendly interfaces

### Topic 1.7: CSS Features
**Difficulty:** Easy | **Tag:** skill  
*Variables, transitions, transforms, animations intro.*

- CSS custom properties (variables)
- Transitions and timing functions
- Transforms (translate, scale, rotate, skew)
- Opacity and filters
- Gradients and shadows

## Learning Path Tips

1. **Build Projects**: Create real projects at each phase
   - Phase 1: Personal portfolio with HTML/CSS
   - Phase 2: Interactive to-do app with vanilla JS
   - Phase 3: Weather app with API integration
   - Phase 4: Todo app or blog with React
   - Phase 5: Full-stack application with Next.js

2. **Practice Incrementally**: Don't skip phases
   - Strong JS fundamentals make React much easier
   - Understand vanilla DOM manipulation before learning React abstraction

3. **Read Code**: Study popular libraries and frameworks
   - Read open-source React code on GitHub
   - Understand design patterns in real projects

4. **Stay Updated**: Frontend evolves rapidly
   - Follow React blog and release notes
   - Join communities (React subreddit, Discord servers)
